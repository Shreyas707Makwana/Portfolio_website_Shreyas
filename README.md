# My Portfolio (Static)

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build & Preview

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push to GitHub.
2. In Vercel dashboard, import this repo.
3. Vercel will detect `vercel.json`:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
